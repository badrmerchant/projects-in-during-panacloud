HTML5 is the Present

A lot of people still think that HTML5 is the future. I think that HTML5 is here right now. I have put together this small web app to demonstrate the capabilities of HTML5 and what you can do with it. Just like Facebook and Gmail this is a single page web app, thanks to the magic of AngularJS. In HTML5 is equally comfortable in creating either 2D vector graphics or jaw dropping 3D graphics. It is now even possible to create real-time communication web apps with WebRTC and Web Sockets. Let us innovate and build some jaw dropping awesome apps using this new technology right now.
